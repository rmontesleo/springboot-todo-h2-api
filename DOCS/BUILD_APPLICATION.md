# Build Application

[RETURN TO README](../README.md)

---

### Ask for help
```bash
./mvnw --help
```

### Clean target
```bash
./mvnw clean
```

### Compile project
```bash
./mvnw compile
```

### Run test
```bash
./mvnw test
```

### Run project
```bash
./mvnw spring-boot:run
```

### Package project
```bash
./mvnw package
```

### run with java file
```bash
java -jar target/springboot-todo-h2-api.jar 
```
